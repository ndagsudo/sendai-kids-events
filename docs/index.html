仙台・宮城の子どもイベント

仙台市・宮城県周辺の親子・小学生向けイベントを掲載しています。
今週末（7月18日(土)・7月19日(日)）のおでかけにぜひ活用してください。

科学体験・工作・自然観察・スポーツ・ワークショップなど、
子どもが楽しめるイベントを毎朝自動更新でお届けします。
無料イベントも多数掲載中！お気に入りを見つけてください。

掲載中の施設・会場

  • 仙台市科学館（青葉区） — 科学実験・ものづくり体験など、親子で楽しめるイベントを定期開催。
  • イオンモール仙台上杉（青葉区） — 大型ショッピングモール内で開催されるキッズワークショップや体験イベント。
  • 宮城県こもれびの森 森林科学館（川崎町） — 豊かな自然の中で竹工作・苔玉作りなど季節の体験教室を実施。
  • 仙台市縄文の森広場（太白区） — 縄文時代の暮らしを体験できる施設。土器・石器体験など子ども向け企画が充実。
  • JRフルーツパーク仙台あらはま（若林区） — 果物の収穫体験・農業体験ができる公園型施設。季節ごとのイベントあり。

※ 掲載施設は順次追加予定です。
